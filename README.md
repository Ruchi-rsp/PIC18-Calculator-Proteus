PIC18 Microcontroller-Based Calculator (Proteus Simulation)

Overview
This project implements a basic calculator using a PIC18 microcontroller, simulated in Proteus.

Objective
To design and simulate a microcontroller-based calculator for performing arithmetic operations.

Features
- Addition
- Subtraction
- Multiplication
- Division
- Simulation-based verification

Technologies Used
- PIC18 Microcontroller
- MPLAB C18
- Proteus Simulation

Working Principle
User inputs are processed by the PIC18 microcontroller, which performs arithmetic operations and displays results.


- Microcontroller interfacing
- Embedded C programming
- Proteus simulation
